The `math.j0()` function returns the order-zero Bessel function of the first kind.

_**Output data type:** Float_

```js
import "math"

math.j0(x: 1.23)

// Returns 0.656070571706025
```

## Parameters

### x
The value used in the operation.

_**Data type:** Float_
